# PyBer_Analysis

## Overview
The purpose of this analysis is to summarize H1-2019 ride sharing data between Rural, Urban and Suburban cities including
- Number of rides
- Number of drivers
- Total fares
- Average fare per ride and driver
- Trends of fares collected over time
